# ManiaUtilities
A variety of osu!mania utilities bundled into a single cross-platform application. 

This project is intended to replace CollectorFLNConverter offering more utilities for beatmap conversion while maintaining the convenience of osu memory reading.

Planned features include:
- Working on Windows and Linux (Mac impossible as far as I now at the moment due to process reading being more locked down on Mac, however if I discover a way, the switch to Avalonia for UI will allow me to quickly patch it in)
- Beatmap to FLN
- Remove SV
- Remove all LNs
- Remove only short LNs
- Rate changing 
- Bulk conversions
- Difficulty override for HP and OD
- Column swapping (including converting lower keycounts to higher keycounts)
- and possibly more if get more ideas
